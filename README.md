# Free Email Extractor

Free Email Extractor is a simple tool that extracts email addresses from websites.  
The system runs with a backend hosted on a VPS and works 24/7.

## Features

- Extract emails from multiple URLs
- Fast processing
- Simple interface
- Backend running on VPS (24/7)
- Easy to use

## How to Use

1. Download the **index.html** file from this repository.
2. Open the file in your browser.
3. Upload or paste the **URLs** you want to scan.
4. Click **Start** to begin email extraction.

## Requirements

- Modern web browser
- Internet connection
- Access to the backend API (hosted on VPS)

## Backend

The backend service runs on a **VPS server 24/7**, handling all email extraction and processing.
